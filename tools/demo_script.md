Demo Script — VibeCalendar (2–3 minute live demo)

1) Open app and start demo
- Click `Start full demo` on the sign-in screen (auto-logins as Alice).
- Say: “We’ll show the core loop: connect a calendar, surface the best next action, and keep private details private.”

2) Today (Step 1)
- Show the Morning Summary and the top suggestion.
- Click `Refresh suggestions` to run the engine live.
- Say: “Suggestions are gap-first, effort-aware, and respect your privacy settings.”

3) Calendar (Step 2)
- Press `Next` to move to Calendar tab (or switch manually).
- Show connected calendars; point out any hidden imports.
- Click `Sync now` on the iCal provider to show import flow.
- Say: “Hidden imports create busy blocks but never expose event details.”

4) Settings (Step 3)
- Press `Next` to move to Settings.
- Toggle location/social/write-back consent and say: “Users are in control — write-back is opt-in.”

5) Finish
- Press `Next`/Finish to end the walkthrough; invite questions.

Notes
- Use the demo accounts if you want different cases: Alice (mixed day), Bob (hidden busy blocks), Carla (overnight work).
- If you need a fully single-screen demo, enable `Compact demo` in the demo banner to hide non-essential tabs.
